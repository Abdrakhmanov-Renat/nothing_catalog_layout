# Getting Started

1. Clone the repository:
```bash
  git clone https://github.com/Abdrakhmanov-Renat/nothing_catalog_layout.git
  cd nothing_catalog_layout
```
2. Install dependencies:
```bash
npm install
# or
yarn install
```
3. Run the project locally:
```bash
npm start
# or
yarn start
```
